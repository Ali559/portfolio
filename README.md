## 🛠️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/ali559/portfolio.git
cd portfolio
```

### Install Dependencies

```bash
npm install
# or
pnpm install
```

### Run the Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
